<b>Audile</b> - приложение для распознавания музыки, которое может помочь вам быстро и точно распознать музыкальный трек, играющий поблизости.
Это приложение использует удаленные сервисы AudD и ACRCloud в качестве API для распознавания музыки, а также сервис Odesli для получения дополнительных ссылок на музыкальные сервисы.

<b>Возможности:</b>

<b>Распознавание</b> - Audile позволяет вам распознать песню в один клик. Приложение сохранит запись, если нет подключения к интернету, и процесс распознавания будет выполнен, когда вы снова подключитесь к сети. Вы можете настроить поведение при неудачном распознавании (ошибка интернета, нет совпадений, другая ошибка) через настройки.
<b>Использование в фоновом режиме</b> - Используйте приложение из панели уведомлений, виджета на главном экране или с помощью плитки быстрых настроек. Управляйте процессом распознавания и получайте информацию о треке без запуска приложения.
<b>Информация о треке</b> - В случае успешного распознавания предоставляется информация о треке: название, исполнитель, альбом и год выпуска, обложка, ссылки на этот трек в популярных музыкальных сервисах, текст песни.
<b>Библиотека</b> - Все распознанные треки хранятся в библиотеке приложения. Вы можете фильтровать список треков, создавать коллекцию избранных, искать треки, удалять их.
<b>Настройки</b> - В приложении есть набор опций для настройки его поведения и внешнего вида.
